🔧 كيفية الاستخدام / HOW TO USE:

الخطوة 1️⃣: تشغيل البرنامج

الخطوة 2️⃣: تبديل اللغة

الخطوة 3️⃣: الاستخدام العادي
   ├─ اختر صورة (بالعربية أو الإنجليزية)

   └─ احفظ النتیجة

💻 أمثلة البرمجة / CODE EXAMPLES:

═══════════════════════════════════════════════════════════════════════════════════

مثال 1️⃣: الحصول على نص مترجم
   from i18n import tr

   text = tr("app_title")
   # يوضع "Iconora Studio" في كلا اللغتين
   print(text)  # نفس النص بالعربية + الإنجليزية

مثال 2️⃣: نصوص مع متغيرات
   from i18n import tr

   msg = tr("msg_image_info", width=1024, height=768, format="PNG")
   # العربية: "الحجم: 1024x768 | الصيغة: PNG"
   # English: "Size: 1024x768 | Format: PNG"

مثال 3️⃣: تبديل اللغة برمجياً
   from i18n import set_language, tr

   set_language("ar")  # تعيين العربية
   print(tr("btn_select_image"))  # "📁 اختر صورة"

   set_language("en")  # تعيين الإنجليزية
   print(tr("btn_select_image"))  # "📁 Select Image"

مثال 4️⃣: في رسائل الخطأ
   from i18n import tr
   from tkinter import messagebox

   try:
       # كود ما
   except Exception as e:
       messagebox.showerror(
           tr("msg_error"),
           f"{tr('msg_load_error')}{e}"
       )

มثال 5️⃣: إضافة ترجمة جديدة
   # 1. أضف في i18n.py:
   TRANSLATIONS = {
       "your_key": {
           "ar": "النص بالعربية",
           "en": "Text in English"
       }
   }

   # 2. استخدمها:
   text = tr("your_key")
