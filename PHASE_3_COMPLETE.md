## 🎨 أمثلة الكود

### مثال 1: إنشاء شعار بسيط
```python
from core.logo_engine import LogoEngine

engine = LogoEngine()
success, msg, path = engine.generate_logo(
    text="My Company",
    font_name="arial.ttf",
    font_size=100,
    text_color="#FFFFFF",
    bg_color="#003366"
)
print(f"Logo created at: {path}")
```

### مثال 2: شعار بتدرج لوني ممتاز
```python
success, msg, path = engine.generate_logo(
    text="Premium Brand",
    font_size=120,
    use_gradient=True,
    gradient_color2="#FF6B6B",
    gradient_direction="diagonal",
    effects={
        "shadow": True,
        "blur": False
    }
)
```

### مثال 3: إنشاء دفعات من الشعارات
```python
logos_data = [
    {"text": "Company A", "text_color": "#FFF", "bg_color": "#0066FF"},
    {"text": "Company B", "text_color": "#000", "bg_color": "#FF9933"},
    {"text": "Company C", "text_color": "#FFF", "bg_color": "#33CC33"},
]
results = engine.batch_generate_logos(logos_data)

for logo in results['success']:
    print(f"✓ {logo['text']}: {logo['path']}")
```
