## 📱 نظام الترجمة (Localization System)

تم بناء نظام ترجمة شامل يدعم العربية والإنجليزية في كل جوانب التطبيق.

### 🗂️ المكونات الرئيسية

#### 1. ملف الترجمة (`i18n.py`)
```python
from i18n import tr, set_language, get_language

# الحصول على نص مترجم
text = tr("app_title")  # "Iconora Studio"

# تعيين اللغة
set_language("ar")  # العربية
set_language("en")  # English

# الحصول على اللغة الحالية
lang = get_language()  # "ar" or "en"
```

### 🌍 الاستخدام

#### في الواجهات (UI)
```python
from i18n import tr

# استخدام نص مترجم
label = ctk.CTkLabel(
    frame,
    text=tr("btn_select_image")  # سيكون "📁 اختر صورة" بالعربية
)
```

#### مع المتغيرات
```python
# نص بمتغيرات
message = tr("msg_saved_images", count=5)
# يصبح: "تم حفظ 5 صور بنجاح"

info = tr("msg_image_info", width=800, height=600, format="PNG")
# يصبح: "الحجم: 800x600 | الصيغة: PNG"
```

#### في مربعات الرسائل
```python
messagebox.showinfo(
    tr("msg_success"),
    tr("msg_saved_ico")
)
```

## 🔧 إضافة ترجمات جديدة

### الخطوة 1: أضف المفتاح إلى `i18n.py`

```python
TRANSLATIONS = {
    "your_new_key": {
        "ar": "النص بالعربية",
        "en": "Text in English"
    },
    # ... باقي المفاتيح
}
```

### الخطوة 2: استخدمه في الكود

```python
from i18n import tr

text = tr("your_new_key")
messagebox.showinfo("عنوان", text)
```

### الخطوة 3: مع المتغيرات (اختياري)

```python
TRANSLATIONS = {
    "msg_custom": {
        "ar": "الرسالة مع {variable}",
        "en": "Message with {variable}"
    }
}

# الاستخدام
msg = tr("msg_custom", variable="قيمة")
```
