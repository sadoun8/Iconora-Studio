🚀 كيفية الاستخدام / HOW TO USE:

1. تشغيل البرنامج:
   python main.py

2. تبديل اللغة:
   - انقر على زر "العربية 🇸🇦" أو "English 🇬🇧" في أعلى النافذة
   - ستتم إعادة تحميل الواجهة فوراً باللغة الجديدة

3. في الكود:
   from i18n import tr
   text = tr("your_key")
   # يحصل على النص المترجم حسب اللغة الحالية

📚 الأمثلة / EXAMPLES:

مثال 1: استخدام بسيط
    from i18n import tr
    message = tr("app_title")
    # العربية: "Iconora Studio"
    # English: "Iconora Studio"

مثال 2: نصوص مع متغيرات
    from i18n import tr
    msg = tr("msg_image_info", width=1024, height=768, format="PNG")
    # العربية: "الحجم: 1024x768 | الصيغة: PNG"
    # English: "Size: 1024x768 | Format: PNG"

مثال 3: تبديل اللغة
    from i18n import set_language
    set_language("ar")  # العربية
    set_language("en")  # English

مثال 4: في رسائل الخطأ
    from i18n import tr
    from tkinter import messagebox

    messagebox.showerror(
        tr("msg_error"),
        tr("msg_load_error")
    )
