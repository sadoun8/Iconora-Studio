## 🔄 تحميل دفعة من الشعارات:

```python
from core.logo_engine import LogoEngine

engine = LogoEngine()

# أنشئ عدة شعارات
logos = [
    {"text": "Company A", "text_color": "#FFF", "bg_color": "#0066FF"},
    {"text": "Company B", "text_color": "#FFF", "bg_color": "#FF0000"},
    {"text": "Company C", "text_color": "#FFF", "bg_color": "#00AA00"},
]

results = engine.batch_generate_logos(logos)

# النتائج:
for logo in results['success']:
    print(f"✓ {logo['text']}: {logo['path']}")
```
