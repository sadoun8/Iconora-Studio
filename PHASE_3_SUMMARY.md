## 📝 أمثلة الاستخدام

### مثال 1: إنشاء شعار بسيط
```python
from core.logo_engine import LogoEngine

engine = LogoEngine()
success, msg, path = engine.generate_logo(
    text="My Company",
    font_name="arial.ttf",
    font_size=100,
    text_color="#FFFFFF",
    bg_color="#003366"
)
```

### مثال 2: شعار بتدرج لوني
```python
success, msg, path = engine.generate_logo(
    text="My Company",
    font_size=100,
    use_gradient=True,
    gradient_color2="#FF6B6B",
    effects={"shadow": True}
)
```

### مثال 3: إنشاء عدة شعارات
```python
logos_data = [
    {"text": "Company A", "text_color": "#FFFFFF", "bg_color": "#0066FF"},
    {"text": "Company B", "text_color": "#000000", "bg_color": "#FF9933"},
    {"text": "Company C", "text_color": "#FFFFFF", "bg_color": "#33CC33"},
]
results = engine.batch_generate_logos(logos_data)
```
